4. int 타입의 파라미터를 2개받아와서 int 타입의 뺄셈의 결과를 return해주는 메서드를 생성 한다.(5점)

(테이블ㆍ인덱스ㆍ제약조건을 이해하고 데이터베이스 설계에 따라 설치된 DBMS에 적합한 오브젝트 생성 계획을 수립할 수 있다.)
